# Agents League Hackathon @ AI Skills Fest 2026

10-day Microsoft developer competition: live coding battles + asynchronous challenges + community. 4,224 registrants, 1,136 projects (as of registration page snapshot). 509 projects linked to the Reasoning Agents challenge.

## Key dates

| Date | Milestone |
|------|-----------|
| May 19 – June 14 | Official entry period (rules) |
| June 4 | Submissions open |
| June 8–12 | AISF Event Week |
| Wed June 10, 9 AM PT | Reasoning Agents live battle (Microsoft Reactor) |
| **June 14, 11:59 PM PT** | **Submission deadline** |
| June 30 | Winners announced |

⚠️ Today is **June 12, 2026** → roughly **2.5 days** to deadline.

## Tracks

| Track | Tool | Challenge |
|-------|------|-----------|
| 🎨 Creative Apps | GitHub Copilot | Creative applications via AI-assisted dev |
| 🧠 **Reasoning Agents** ← ours | **Microsoft Foundry** | Multi-step reasoning agents |
| 💼 Enterprise Agents | Microsoft 365 Copilot | Business-ready M365 agents |

## Hard requirement: Microsoft IQ

**All submissions must integrate at least one Microsoft IQ layer** — Foundry IQ, Work IQ, or Fabric IQ. Any IQ works in any track. There is a separate **"Best Use of IQ Tools" prize ($6,468 FMV)**, so deep IQ integration is double-rewarded.

## Prizes (FMV)

- Best Overall Agent — $16,468 (incl. $15k cash)
- Best Reasoning Agent — $6,468 (incl. $5k cash)
- Best Use of IQ Tools — $6,468 (incl. $5k cash)
- Top Student / Accessibility / Hack for Good — $1,468 each (3 winners each; student award requires enrolled university students)
- All submitters get a digital badge (issued by Global AI Community)
